# MyCompany 1.5.137

Data: 2026-07-23

## Cel

Ujednolicenie podobnych widoków SirK Portal na jednym kontrakcie UI oraz usunięcie starych skryptów i arkuszy, które nadpisywały się wzajemnie.

## Najważniejsze zmiany

- dodano jeden końcowy kontrakt CSS i JavaScript dla wszystkich modułów;
- ujednolicono shell, toolbar, trzy kolumny, nawigację, karty, formularze i tabele;
- ustalono wspólne szerokości `184 / 236 / 440 / 56 px`;
- Toolbar, katalog i status navigation emitują bezpośrednio wspólne klasy `mc-portal-*`;
- Zarządzanie i Ustawienia korzystają z tego samego rzeczywistego gridu co pozostałe moduły;
- osadzone Ustawienia otrzymują końcowy CSS, akcent i jasny/ciemny Portalu;
- usunięto z `portal-device-tabs.css` wszystkie reguły niezwiązane z Devices;
- usunięto stary globalny mutator `portal-standalone-nav.js`, który przenosił elementy sesji, usuwał klasy Zarządzania, wstrzykiwał CSS i stosował timeout 15 sekund;
- zachowano wyłącznie lokalne porządkowanie kompaktowego nagłówka Devices;
- browser asset version został zrównany z `1.5.137`;
- dodano `test/portal-ui-contract.test.js` oraz włączono go do `npm test`.

## Kontrola po wdrożeniu

1. sprawdź Polecenia, Zarządzanie, Akceptacje i Ustawienia;
2. zwiń i rozwiń pierwszą kolumnę każdego dostępnego modułu;
3. włącz Edit Mode i sprawdź poszerzenie drugiej kolumny;
4. porównaj toolbar, aktywne zaznaczenie, karty, formularze i tabele;
5. przełącz PL/EN oraz jasny/ciemny;
6. sprawdź, czy Devices nadal zachowuje aktywne sesje i kompaktowy nagłówek;
7. wykonaj `F5` na aktywnym hoście i w każdym głównym widoku.

## Restart

Zmiana zawiera nowe i zmodyfikowane assety przeglądarki oraz zmianę numeru wersji. Po instalacji wymagany jest restart MeshCentral i pojedyncze `Ctrl+Shift+R`.
